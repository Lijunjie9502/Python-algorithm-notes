
# 15. 二进制中 1 的个数

[LeetCode](https://leetcode-cn.com/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/)

## 题目描述

输入一个整数，输出该数二进制表示中 $1$ 的个数。

#### $n\&(n-1)$

该位运算去除 $n$ 的位级表示中最低的那一位。

```html
n       : 10110100
n-1     : 10110011
n&(n-1) : 10110000
```

时间复杂度：$O(M)$，其中 $M$ 表示 $1$ 的个数。


```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        res = 0
        while True:
            if n == 0: break
            res += 1
            n &= n - 1
        return res
```

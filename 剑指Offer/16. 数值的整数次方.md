# 16. 数值的整数次方

[LeetCode](https://leetcode-cn.com/problems/shu-zhi-de-zheng-shu-ci-fang-lcof/)

## 题目描述

给定一个 double 类型的浮点数 base 和 int 类型的整数 exponent，求 base 的 exponent 次方。

### 解题思路

下面的讨论中 $x$ 代表 $base$，$n$ 代表 $exponent$

$$x^{n}=\left\{\begin{array}{ll}
(x * x)^{n / 2} & n \% 2=0 \\
x *(x * x)^{n / 2} & n \% 2=1
\end{array}\right.$$



因为 $(x*x)^{n/2}$ 可以通过递归求解，并且每次递归 $n$ 都减小一半，因此整个算法的时间复杂度为 $O(\log N)$。

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n == 0: return 1
        if n == 1: return x
        if n == -1: return 1/x
        myPow_2 = self.myPow(x, n // 2)
        return myPow_2 * myPow_2 if n % 2 == 0 else myPow_2 * myPow_2 * x
```
